# Description

project that introduces the fundamentals of using PostgreSQL with database operations. This demo focuses on understanding how to create and manage databases, tables, and perform common SQL queries.

# The project includes examples of:

Creating a PostgreSQL database

Creating tables with different data types

Inserting records into tables

Retrieving data using SELECT queries

Updating existing records

Deleting records from tables

Understanding primary keys and constraints

Practicing basic CRUD operations

This demo is designed for beginners who want to learn how PostgreSQL works and how to interact with it using SQL commands.
